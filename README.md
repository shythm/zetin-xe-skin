# ZETIN XE Skin

- ZETIN 홈페이지를 위한 스킨
- XE(XpressEngine 1) CMS를 위한 스킨
- HTML, CSS, JS를 적절히 잘 활용함
- 반응형 웹 페이지
- 군대 가기 전에 심심해서 만듦(2020년도 ZETIN 웹 마스터가 되면서 홈페이지를 좀 꾸며봐야겠다는 생각도 들었었음)
- 아직 개발중 ...
